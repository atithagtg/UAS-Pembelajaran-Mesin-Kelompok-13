## Judul Proyek
Prediksi Konsumsi Energi Bangunan Menggunakan Random Forest dan XGBoost untuk Mendukung Efisiensi Energi dalam Transisi Energi 

---

## Deskripsi Proyek
Project ini bertujuan membangun model prediksi konsumsi energi bangunan menggunakan metode supervised learning dengan pendekatan regresi

Model yang digunakan:
- Random Forest Regressor
- XGBoost Regressor

Evaluasi model dilakukan menggunakan:
- MAE
- RMSE
- RMSLE
- R²

## Dataset
Dataset yan digunakan berasal dari:

*Kaggle, ASHRAE - Great Energy Predictor III*
**https://www.kaggle.com/c/ashrae-energy-prediction**
